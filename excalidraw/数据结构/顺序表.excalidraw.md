public DynamicArray(int initialCapacity) {
    if (initialCapacity > 0) {
        elementData = new Object[initialCapacity];
    } else if (initialCapacity == 0) {

DEFAULT_CAPACITY_EMPTY_ELEMENT_DATA 与 EMPTY_ELEMENT_DATA 虽然都是空数组，但是有所不同，主要体现在添加第一个元素后数组扩容后的容量大小不一样。
对于无参构造，elementData = DEFAULT_CAPACITY_EMPTY_ELEMENT_DATA，在添加第一个元素之后，数组扩容后的容量大小会变为 DEFAULT_CAPACITY = 10；
对于有参构造，如果指定初始容量 = 0，elementData = DEFAULT_CAPACITY_EMPTY_ELEMENT_DATA，在添加第一个元素之后，数组扩容后的容量大小会变为 1; ^HCpJga2D

public void add(int index, E e) {

private Object[] grow() {
    return grow(size + 1);
}

private Object[] grow(int minCapacity) {
    int oldCapacity = elementData.length;
    if (oldCapacity > 0 || elementData != DEFAULT_CAPACITY_EMPTY_ELEMENT_DATA) {
        int newCapacity = oldCapacity + Math.max(minCapacity - oldCapacity, oldCapacity >> 1);
        System.out.printf("grow: oldCapacity = %d => newCapacity = %d%n", oldCapacity, newCapacity);
        return elementData = Arrays.copyOf(elementData, newCapacity);
    } else {
        int newCapacity = Math.max(DEFAULT_CAPACITY, minCapacity);
        System.out.printf("grow: oldCapacity = %d => newCapacity = %d%n", oldCapacity, newCapacity);
        return elementData = new Object[newCapacity];
    }
} ^XdRh1vPu

2.1 确定新容量大小：
    如果 elementData = DEFAULT_CAPACITY_EMPTY_ELEMENT_DATA，则新容量大小 = 10；
    如果 elementData = EMPTY_ELEMENT_DATA，则新容量大小 = 1；
    如果旧容量 > 0，则新容量 = 旧容量 * 1.5； ^lu65dZ4k

2.2 数据迁移：确定好新容量大小之后，创建一个新的数组，将原数组中的所有元素全部拷贝到新数组上 ^ftyf6oMV

4. 最后使用待添加元素的值覆盖指定位置上的元素即可，元素个数加一 size++； ^uqL3v4iL

3. 在指定的位置添加元素，则需要将指定位置之后的所有元素全部都往后挪一格，该如何实现呢？两种方式：

        for (int i = size; i > index; i--) {

线性表：由 n（n >= 0）相同类型的数据元素组成的有限序列，它是最基本、最常用的一种线性结构。 ^aEecVtkC

当i = 1,2,...,n-1时，ai有且仅有一个直接后继，当i=2,3,...,n时，ai有且仅有一个直接前驱。
⬇️
除第一个元素外，每个元素都有唯一的直接前驱；除最后一个元素外，每个元素都有唯一的直接后继。 ^evZDCt0q

表中ai-1领先于ai，ai领先于ai+1，称ai-1是ai的直接前驱元素，ai+1是ai的直接后继元素。 ^MuFuwgaM

线性表中元素的个数n（n >= 0）定义为线性表的长度，当n = 0 时称为空表。 ^DCcKGh0m

在非空表中的每个数据元素都有一个确定的位置，如a1是第一个数据元素，an是最后一个数据元素，
ai是第i个数据元素，称i为数据元素ai在线性表中的位序。 ^Yc2K63Dv

有两种储存方式：顺序存储和链式存储。采用顺序存储的线性表被称为顺序表，采用链式存储的线性表被称为链表。 ^NMayog0n
