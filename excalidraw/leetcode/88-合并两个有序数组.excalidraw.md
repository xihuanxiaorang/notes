public void merge(int[] nums1, int m, int[] nums2, int n) {
    int i = 0, j = 0, k = 0;
    int[] temp = new int[m + n];
    while (i < m && j < n) {
        if (nums1[i] < nums2[j]) {
            temp[k++] = nums1[i++];
        } else {
            temp[k++] = nums2[j++];
        }
    }
    while (i < m) {
        temp[k++] = nums1[i++];
    }
    while (j < n) {
        temp[k++] = nums2[j++];
    }
    for (int l = 0; l < m + n; l++) {
        nums1[l] = temp[l];
    }
} ^Fwi1AMMb

public void merge(int[] nums1, int m, int[] nums2, int n) {
    int i = m - 1, j = n - 1, k = m + n - 1;
    while (i >= 0 && j >= 0) {
        if (nums2[j] > nums1[i]) {
            nums1[k] = nums2[j--];
        } else {
            nums1[k] = nums1[i--];
        }
        k--;
    }
    while (j >= 0) {
        nums1[k--] = nums2[j--];
    }
} ^IxYwAjip

思路二：由于两个数组中的元素都是从小到大排序的，所以可以先从数组后面开始比较，也就需要倒序遍历两个数组，依次判断这两个数组中哪个的值更大，
        假如 nums1[i] 的值 > nums2[j] 的值，则 nums1[k--] 就等于 nums1[i--] 的值，反之 nums1[k--] 就等于 nums2[j--] 的值，
        其中 i 的初始值为 m - 1，j 的初始值为 n - 1，k 的初始值为 m + n - 1；当 i >= 0 && j >= 0 的时候，就重复上述比较的过程。
        最后，判断 nums2 数组中的元素是否没有遍历完，如果没有遍历完，则继续遍历，将 nums2 数组中剩余的元素倒着依次放入 nums1 数组中；
